[*] Introduction:
=============

 HANGMANbit is the typical game in which you will have to complete words using letters.
 If you make a mistake in some letters you will add a part of the skeleton and if the
 skeleton is complete you will have lost.


=========
[*] Controls:
=========

 The keyboard letters :-)
